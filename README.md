Project Title: Texting / Chat UI Generator
Description: A React application to generate realistic chat mockups for iMessage, WhatsApp, and Snapchat.
Features:
Support for multiple chat styles (iMessage, WhatsApp, Snapchat).
Customizable message content.
Video generation capabilities (based on output folder and previous context).
Installation: npm install
Usage: npm run dev
